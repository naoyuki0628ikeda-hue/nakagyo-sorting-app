# 京都支店仕分けアプリ（シンプル現場版）

## 使い方（現場の流れ）
1. **区を選択**
2. **郵便番号を入力**
   - 基本：**下4桁**（例：600-8098 → 8098）
   - 7桁（例：6008098）でもOK（区は未選択でも検索可）
3. **仕分けCD** が表示されます（表示をタップでコピー）

## 注意
- 下4桁検索は、区を選ばないと判別できないため **区の選択が必須**です。
- 同じ下4桁で複数候補がある場合でも、仕分けCDが共通ならそのまま表示します。
  仕分けCDが複数に分かれる場合は「要確認」と出ます。7桁入力で検索してください。
