# 中京区仕分け用アプリ（レイアウト版）

## 起動
このフォルダでコマンドプロンプトを開いて実行：
```bat
python -m http.server 8000
```
ブラウザで：
- http://localhost:8000
（一覧が出る場合は http://localhost:8000/index.html ）

## 使い方
- 伝票の「郵便番号 下5ケタ」を入力 → 仕分けコード（4桁）を表示
- 同じ下5ケタが複数候補の場合は、候補をタップして選択
- 仕分けコード部分をタップするとコピー

## データ更新（Excel→JSON）
Excelを更新したらこのフォルダで：
```bat
python build_data.py "仕分け番号（中京区）_郵便番号入力済.xlsx"
```
