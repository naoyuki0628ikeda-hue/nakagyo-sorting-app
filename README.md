# 京都支店仕分けアプリ（静的Webアプリ）

このフォルダをそのまま静的ホスティング（例：GitHub Pages / Netlify）に置くだけで動きます。

## 使い方
- **住所**：スペース区切りでAND検索（例：`中京区 西ノ京`）
- **郵便番号**：`604-xxxx` や `604` のように入力（短いと前方一致）
- **仕分けコード**：`0403` や `04` のように入力（短いと前方一致）

## ファイル
- `index.html` 画面
- `app.js` 検索ロジック
- `data.json` 仕分けコードデータ（Excelから生成）

## GitHub Pages の例
1. GitHub リポジトリの `main` か `gh-pages` ブランチに、このフォルダ中身をアップロード
2. Settings → Pages → Branch を選んで公開
