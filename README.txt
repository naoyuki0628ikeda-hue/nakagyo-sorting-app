# 京都 支店仕分け（Webアプリ）

## できること
- 区を選択 → 郵便番号の下4〜5桁（または7桁）を入力すると、仕分けCDを表示します
- 入力と同時に自動検索（検索ボタン不要）
- 同じ郵便番号でも仕分けCDが複数あり得る場合は「候補」を表示して選択できます

## 使い方（現場向け）
1. 区をタップ
2. 伝票の郵便番号の下4〜5桁を入力（または7桁でもOK）
3. 仕分けCDが表示されます

## ファイル構成
- index.html
- style.css
- app.js
- data.js（仕分CD.xlsx から自動生成したデータ）

## 更新方法（データ差し替え）
1. 仕分CD.xlsx を更新
2. この生成スクリプト（ChatGPT）で data.js を作り直して同じ場所にアップロード
   - もしくは再度「仕分CD.xlsx を元に更新版を作って」と依頼してください

## QRコードについて
- GitHub Pages 等で公開したURLをQRコード化して、現場に掲示すると運用しやすいです
